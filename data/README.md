# Dataset: UCI Bank Marketing (Social/Economic Context)

## Source
- **Original paper:** S. Moro, P. Cortez and P. Rita. *A Data-Driven Approach to Predict the Success of Bank Telemarketing.* Decision Support Systems, 2014. https://doi.org/10.1016/j.dss.2014.03.001
- **UCI repository:** https://archive.ics.uci.edu/ml/datasets/Bank+Marketing
- **Economic indicators:** Banco de Portugal — https://www.bportugal.pt/estatisticasweb

## File in Use
`raw/bank-additional-full.csv` — 41,188 rows · 21 columns · semicolon-delimited  
Period: May 2008 – November 2010 (Portuguese retail bank telemarketing campaign)

## Target Variable
| Column | Type | Values |
|--------|------|--------|
| `y` | Binary | `"yes"` = subscribed to term deposit · `"no"` = did not subscribe |

**Class imbalance:** ~11 % positive (`yes`) — requires stratified splits and imbalance-aware modelling (SMOTE / class weights).

## Missing Values
Several categorical columns use `"unknown"` as a missing-value label.  
Strategy (see `config/config.yaml`): treat `unknown` as its own category for tree-based models; impute for linear models.
